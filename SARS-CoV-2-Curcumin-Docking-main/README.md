# 🧬 SARS-CoV-2-Curcumin-Docking

An in-silico molecular docking analysis evaluating the binding potential of **Curcumin** against the **SARS-CoV-2 Main Protease ($M^{pro}$)** to calculate the thermodynamic stability of the complex. This project demonstrates Curcumin's ability to anchor into the catalytic cleft using computational structural biology tools.

## ✨ Key Parameters & Results

| Parameter | Details |
|---|---|
| **Target Protein** | SARS-CoV-2 Main Protease ($M^{pro}$) — **PDB: 6LU7** |
| **Ligand** | Curcumin — **CID: 969516** |
| **Top Binding Affinity** | **-5.731 kcal/mol** |
| **Interaction Summary** | Curcumin successfully positions its aromatic rings and hydroxyl groups to form stable polar contacts within the $M^{pro}$ active site. |

---

## 🛠️ Tools & Pipeline

| Component | Technology / Platform |
|---|---|
| **Target Preparation** | RCSB PDB, PyMOL (for removal of water and heteroatoms) |
| **Ligand Preparation** | PubChem |
| **Docking Engine** | AutoDock Vina (via Webina) |
| **Visualization** | PyMOL (Hydrogen bond mapping and polar contacts) |

### 📍 Grid Parameters (AutoDock Vina)
* **Center:** X: -10.5, Y: 12.5, Z: 67.5
* **Size:** 22 × 22 × 22 Å
